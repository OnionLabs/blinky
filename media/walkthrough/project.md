# Create a Project

Start with a ready-made template to get coding faster.

## Available Templates

- **Blink LED** - Toggle the onboard LED. The classic first program.
- **WiFi Scanner** - Scan for WiFi networks and display signal strength.
- **Web Server** - Serve a web page from your board, controllable from a browser.
- **Temperature Sensor** - Read temperature and humidity from a DHT sensor.

## How it works

1. Pick a template
2. Choose a folder for your project
3. The template files (`main.py`, `boot.py`) are created automatically
4. Start editing and upload to your board with **Sync Project**
