# Run a Python File

Write code. Press F5. It runs on your board.

- **F5** - Run the current file on the board
- **Shift+Enter** - Run the current line or selection
- **Shift+F5** - Stop a running script
- **▶ Play button** - Click the play button in the editor title bar

The file is uploaded to the board and executed. Output appears in the REPL terminal, and errors are shown as inline diagnostics (squiggly underlines) in the editor.

> **Tip:** Use Shift+Enter to iterate quickly - the cursor advances to the next line automatically.
